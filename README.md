## Note App - Compose

#### This is and app developed for practice purposes.

##### It makes use of:

* Kotlin
* Jetpack Compose
* Dagger Hilt for dependency Injection
* Room
* Coroutines
* Flow
* Retrofit
* Clean Code
* MVVM
* Repository pattern
* Use Cases
